## About
Tool-53 is a web app that helps security teams track whether their company's security rules are being followed, with an AI layer that automatically describes, categorises, and gives recommendations for each rule.

## Layers of the Application
[ React Frontend ]     ← What user sees and clicks on <br>
       |<br>
[ Spring Boot Backend ] ← Stores data, handles users, sends emails<br>
       |<br>
[ Flask AI Service ]   ← The intelligence layer — Groq + ChromaDB
